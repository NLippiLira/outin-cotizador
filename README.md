# 🛡️ OUTIN Cotizador

Aplicación web profesional para generar cotizaciones de servicios de seguridad CCTV en tiempo real, optimizada para uso en terreno desde dispositivos móviles.

---

## 🚀 Características

- ✅ Generación de cotizaciones dinámicas
- ✅ Cálculo automático de subtotal, IVA (19%) y total
- ✅ Exportación a PDF en formato A4 profesional
- ✅ Diseño optimizado para dispositivos móviles (Mobile First)
- ✅ Integración con WhatsApp para envío rápido al cliente
- ✅ Plantillas rápidas para servicios frecuentes
- ✅ Interfaz simple, rápida y enfocada en ventas

---

## 📱 Uso en terreno

Esta aplicación fue diseñada para uso real en terreno:

1. Ingresar datos del cliente  
2. Agregar servicios rápidamente  
3. Generar PDF  
4. Enviar por WhatsApp en segundos  

---

## 🧾 Tecnologías utilizadas

- React + Vite  
- Bootstrap 5  
- html2canvas  
- jsPDF
